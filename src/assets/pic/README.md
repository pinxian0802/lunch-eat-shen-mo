# 題目圖片說明

## 需要的圖片

請將以下圖片放在此資料夾：

### questionPic.jpg
- **用途**：測驗題目「圖片中的商品叫做什麼名稱」
- **正確答案**：男友寵愛女友組
- **建議尺寸**：建議寬度 400-600px，保持清晰度

### answerPic.jpg
- **用途**：當測驗題目答對時顯示的圖片（替換 questionPic.jpg）
- **適用題目**：「圖片中的商品叫做什麼名稱」
- **建議尺寸**：建議寬度 400-600px，保持清晰度
- **說明**：當使用者選擇正確答案「男友寵愛女友組」時，圖片會自動切換成此圖片
- **格式**：JPG

## 如何添加圖片

1. 將圖片命名為 `questionPic.jpg`
2. 放在 `src/assets/pic/` 資料夾中
3. 確保圖片清晰可見

## 圖片路徑

在程式碼中，圖片路徑為：`/src/assets/pic/questionPic.jpg`

如果圖片無法載入，系統會自動隱藏圖片區域並在 console 顯示錯誤訊息。
